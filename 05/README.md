◎ng→在ALU用來判斷最左邊位元是否為1, 是則為負數(∵二進制),在CPU中功用相當於"<"。(∵若x-y<0 → x<y)

◎zr→在ALU用來判斷全部位元是否為0, 在CPU中功用相當於"="。(∵若x-y=0 → x=y)

此時我們還缺少">"來判斷是否要JUMP,所以要在接出一條線(以下稱為PO)。
◎PO=ng'&zr'---------ng'(小於的相反)相當於 大於 , zr'(等於的相反)相當於大於 或 小於
